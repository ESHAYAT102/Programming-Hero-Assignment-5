Question-1: What are the different ways to select an element in the DOM?

There are a several ways to select an element in the DOM:

1. ID: document.getElementById("id");
2. Class Name: document.getElementByClassName("class");
3. Tag Name: document.getElementByTagName("tag");
4. Query: document.querySelector("query");
document.querySelectorAll("query");

Question-2: What is the difference between innerHTML, innerText, and textContent ?

InnerHTML: Gets or sets the HTML content within an element, including HTML tags.
InnerText: Gets or sets the visible text content of an element, as it would appear on the rendered page.
textContent: Gets or sets the text content of an element, including all child nodes. It returns the raw text, regardless of styling or visibility.

Question-3: What is event delegation in the DOM?

Event delegation leverages event bubbling by attaching a single listener to a parent element, which then handles events originating from its children by checking the "event.target", improving performance and simplifying event management, especially for dynamically added elements.

Question-4: What is event bubbling in the DOM?

In the DOM, event bubbling describes the event's propagation from the point where the event triggered itself, upwards through the element's parent chain (or hierarchy). In short, when you click a nested element that click event will also be detected by its parent, grandparent, etc etc up the document root.

Question-5: How do you create, add, and remove elements using JavaScript?

1. Creating Elements: document.createElement("tagName")
2. Adding Elements: parentElement.appendChild(newElement)
3. Removing Elements: parentElement.removeChild(elementToRemove)
